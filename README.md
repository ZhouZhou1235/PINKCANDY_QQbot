# 粉糖终端

![粉糖终端](/assets/QQ终端.png)

## 描述
周看到有朋友搭建了功能丰富有趣的QQ机器人，这方面还没有经验。
所以参考周的好友白白搭的机叶，设计了一个。
这是一个接入大模型的工具型QQ机器人，可以提供携带记忆的聊天以及实用功能。
机仙，机叶为我编写粉糖终端提供了创意~✨
**具体文档见 /docs**

## 原理
从机器应用账号到QQ服务器之间需要实现通信协议，
机器人框架模拟人类操作QQ，
通过协议到QQ服务器达成整套服务。
