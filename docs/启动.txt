启动


在项目根目录下：


[windows]
点击 launcher.bat 启动napcat协议服务
venv\Scripts\activate
python main.py


[linux]
sudo xvfb-run -a qq --no-sandbox # 启动napcat协议服务
source venv/bin/activate
python3 main.py

用systemd添加服务方便管理
创建服务描述文件 pinkcandy_qqbot.service
systemctl daemon-reload # 重载服务配置
systemctl start <service_name> # 启动
systemctl status <service_name> # 查看状态
systemctl stop <service_name> # 停止
journalctl -u <service_name> # 查看详细日志
